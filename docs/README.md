## About Rob-Bur-

Rob-Bur- is an intelligent robot automation system powered by local LLM (Large Language Models). It combines advanced AI with hardware control for autonomous movement and decision-making.

**Created by**: David Adriano Ferrari dos Santos (CEO)

## Key Features

✨ **AI-Powered Control** - Uses local LLM for intelligent decision making
🚀 **Autonomous Movement** - Step-by-step automated movement with AI guidance  
📊 **Real-time Monitoring** - Track performance and movement in real-time
🔧 **Modular Design** - Easy to extend and customize
